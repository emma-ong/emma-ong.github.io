HTML & CSS explained...Through a house cat's eyes!

HTML - Cat Skeleton

Hypertext Markup Language (HTML) is a language that acts as the foundation of a website’s content and structure, such as how a cat’s skeleton holds its’ innards together and a base to connect its bodily systems.

CSS - Cat Coat

While a cat’s skeleton acts as the foundation for its body, Cascading Style sheets (CSS) acts as a cat’s fur coat. We use CSS to modify the aesthetic and the feel component of a website, as this is what we can use to define schemes of colours, font, alignment and so forth.

Javascript - Cat behaviour

As a programming language, Javascript’s (JS) role is to give a set of instructions of a computer to do something, like how a cat will meow when they might want something from their human or hiss in fear, when they want to be left alone. Instead of telling humans what to do, we are taking control of a computer through providing it with written statements.

Control Flow - Cat routine

A house cat tends to be a creature of routine, sticking to a schedule that has been set either by themselves or their human, and like JS, there is something called a control flow, which describes how a computer runs its code directionally, from top to bottom. A cat might hunt at night and sleep during the day. When JS code runs, it begins from the first line and ends at the last. A routine, however, can be interrupted, such as how a program’s control flow can change. A change can be caused by things called loops or functions, which influence the control flow.

Loops - Cat Naps

Cats are also known to sleep up to 18 hours a day and can be considered lazy in comparison to other animals. When we write code, it is important to harness your inner cat and be as lazy as possible…hold your furballs, let me explain! In programming, we can leverage the computer’s speed to run one or many instruction(s) at a time rather than doing this manually. This is where loops come in handy. For example, if we wanted the computer to display all numbers from 0 to 100, we would write a task for the computer to perform a sequence and to stop running, once it met a certain condition (e.g. once the computer has reached 100).

DOM - Cat VR

While HTML, CSS and JS are declarative in nature, the Document Object Model (DOM) is a means to utilise JavaScript to alter our HTML, which allows us to build websites that are dynamic and display motion or behaviours on the front-end, how a user may navigate within or interact with a website. The DOM represents the actual structure of the page. If HTML and CSS are like a cat’s skeleton and coat, while JS is like a cat’s behaviour, then the DOM can be represented by a dynamic map of a cat’s past, present and plan for its future. Let’s add another metaphor, and imagine the cat was a modifiable character in a virtual world, we would be able to navigate the DOM as though we were in an edit mode in this virtual reality, where we could alter the cat’s state, by changing its appearance or behaviours.

Arrays - List of Fur Friends

A cat is an intelligent being, and like humans, it remembers things. Like humans, they may not remember everything accurately, such as when meeting new humans and other cats. If a cat wanted to keep track of those it comes across, they could use something called an array, which works like a sorted list – the items in this list are in positions called an index. An index is a numeric label of the position of an item in an array. The first item in an array begins with 0 and not 1. The next item on the list will be 1, and then 2 and so on. We can give this list a name to call upon it later in time, this is equivalent to storing the list in a variable and a list is just one of the many things a variable can store, so that we can retrieve it later.
